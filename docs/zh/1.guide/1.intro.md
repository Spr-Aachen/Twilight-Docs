**欢迎使用 Twilight 主题模板!**  

**Twilight** 是使用 **Astro** 框架构建并集成了后台管理系统的开源静态网站主题，能够满足您对于博客、知识库、系列文档等一系列建站需求。

<LinkCard title="👉 去 GitHub 点个 ⭐ 支持一下" href="https://github.com/Spr-Aachen/Twilight">
</LinkCard>


::: collapse
- 不了解[Astro?](https://astro.build)

    Astro 是最适合构建像博客、营销网站、电子商务网站等以内容驱动的网站的 Web 框架。相比其他前端框架，它减少了 JavaScript 的开销和复杂性。如果你需要一个加载速度快、具有良好 SEO 的网站，那么 Astro 就是你的不二之选。
:::


:::tip
本主题目前处于初期阶段。
当前主题功能和 API 尚未稳定，在未来的更新中仍有概率出现破坏更改。
如果您在使用过程中遇到问题，或者有新的想法，请在 Issues 里提出，也欢迎通过 PR 帮助完善主题。
:::
